# Basic instructions for the Repository

This repository contains the code for different parts of Anthropic API course and MCP.
The following folders `api`, `claude_code_app_starter` and `mcp_examples`.
Each of these projects has its own README file with instructions on how to run it, and the dependencies it needs. 

For each project you have to run `uv sync` to create its own environment.
Each project has its own `<project_name>.code-workspace.json` file, which you can open in VS Code to have the right environment and dependencies for that project.
